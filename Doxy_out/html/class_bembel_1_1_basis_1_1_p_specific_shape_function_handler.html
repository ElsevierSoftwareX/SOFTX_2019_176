Detailed Description

template<int P>
class Bembel::Basis::PSpecificShapeFunctionHandler< P >

These routines implement a template recursion that allows to choose a compile time instantiation of a basis-evaluation routine with a runtime p. To replace the underlying basis, only these routines should be changed.
